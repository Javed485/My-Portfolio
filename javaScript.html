JavaScript Projects

A collection of beginner to intermediate level JavaScript projects built to practice DOM manipulation, API usage, and local storage.
This collection is designed specifically with beginners in mind. Each project focuses on a specific concept or technique—such as updating the content of a webpage using the Document Object Model (DOM), retrieving live data from external services using APIs, or saving user data in the browser using localStorage.
No frameworks or libraries were used—just pure —to focus on the fundamentals. Whether it's building a to-do list or working with APIs, each project offers a practical way to apply core concepts and strengthen development skills.

YouTube Clone

A simplified front-end clone of the YouTube platform that replicates key UI elements and functionality. It allows users to browse and search for videos using the YouTube API. This project demonstrates advanced DOM manipulation, API usage, and responsive design skills.

  • HTML
  • CSS
  • JAVASCRIPT
  • YOUTUBE API
  • LOCAL STORAGE
  • LIVE LINK

Hotstar Clone

A front-end clone of the streaming platform interface, designed to showcase movies and TV shows by fetching real-time data from the OMDb API. This project replicates the modern UI/UX of Hotstar and demonstrates dynamic rendering of multimedia content using JavaScript and API calls.

  • HTML
  • CSS
  • JAVASCRIPT
  • OMDB API
  • LOCAL STORAGE
  • LIVE LINK

Add to Cart

A basic e-commerce shopping cart system built using vanilla JavaScript. This project allows users to browse a product list, add items to their cart, and manage quantities. It's a great hands-on exercise in DOM manipulation, event handling and local storage for cart persistence.

  • HTML
  • CSS
  • JAVASCRIPT
  • LOCAL STORAGE
  • LIVE LINK

EMI Calculator

An EMI (Equated Monthly Installment) Calculator that allows users to calculate the monthly installment amount for a loan based on the loan amount, interest rate, and tenure. This is a practical mini-project to apply mathematical logic with DOM manipulation and real-time form inputs.

  • HTML
  • CSS
  • JAVASCRIPT
  • LIVE LINK

Notes App

A simple, interactive Notes App that allows users to create, edit, delete, and save text notes in the browser. The app uses local storage to ensure notes persist even after refreshing or closing the page. This project is ideal for practicing DOM manipulation, event handling and data storage with vanilla JavaScript.

  • HTML
  • CSS
  • JAVASCRIPT
  • LOCAL STORAGE
  • LIVE LINK

Todo App

A To-Do App is a productivity tool that allows users to add, edit, delete, and mark tasks as completed. This project focuses on mastering DOM manipulation, event handling, and local storage to create a fully functional, interactive, and persistent task manager in the browser.

  • HTML
  • CSS
  • JAVASCRIPT
  • LOCAL STORAGE
  • LIVE LINK
