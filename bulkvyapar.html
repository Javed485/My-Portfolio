Bulkvyapar Network B2B Platform

BulkVyapar is a B2B marketplace where retailers source inventory in bulk from vetted suppliers. I worked on the frontend and UI/UX design, focusing on making the platform easy to use, fast, and accessible. I designed and built:

  • Designed user-friendly layouts to make browsing and ordering bulk products simple.
  • Built responsive pages (mobile, tablet, desktop) using Bootstrap grid system.
  • Styled clean product catalogs, supplier profiles, and dashboards with modern UI.
  • Developed interactive features like dropdown menus, modals, and carousels with JavaScript.
  • Focused on UI/UX best practices: consistency, accessibility, and smooth interactions.
  • Ensured fast loading pages with optimized HTML/CSS code.

This collection showcases multiple website design projects I created for BulkVyapar, each focusing on different parts of the platform.
